Texto Excluido e Inserido

Podemos usar a tag del para criar esse efeito de texto excluido, e utilizar a tag ins para dar esse efeito de texto inserido, sublinhado.

Texto sobrescrito e subescrito

Podemos utilizar a tag sup para criar um efeito assim: x2 + 3, observe como o 2 ficou para cima(sublinhado com a tag u, não semântica).

Podemos usar a tag sub para criar um efeito assim:H2O, observe como o 2 ficou para baixo (sublinhado com a tag u, não semântica).
